## ðŸ“š Tentang TypeScript

TypeScript adalah bahasa pemrograman superset dari JavaScript yang menambahkan **pengetikan statis (static typing)** serta fitur modern lain yang membuat pengembangan aplikasi skala menengah-besar menjadi lebih mudah dan terstruktur. ([GitHub][1])

Jenis file utama dalam TypeScript adalah `.ts`, yang kemudian dikompilasi menjadi JavaScript yang dapat dijalankan di lingkungan Node.js atau browser.
